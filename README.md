# eazyCmp 🚀
A powerful, lightweight UI kit for **Compose Multiplatform (Android & iOS)**.

| Feature | Description |
| :--- | :--- |
| 🎭 **Lottie** | Render animations directly from URL. |
| 📅 **DatePicker** | Single & Range selection with constraints (days/millis). |
| 🔽 **Dropdown** | Generic Single/Multi-select with Search & ItemLabel support. |
| 🔘 **Checkboxes** | Fully customizable size (replaces bulky defaults). |
| 📏 **Dashed Divider** | Modern UI dividers for clean layouts. |
| 🏗️ **Bottom Bar** | Generic state-aware navigation bar. |
| ✍️ **TextFields** | Optimized Outlined fields with smart constraints. |
